# docker-flask

## Set up the Flask project with Docker

> Set up environment variable
```bash
export FLASK_APP=<project_file_name>.py

Example:

export FLASK_APP=server.py
```
### Debug Mode

```bash
export FLASK_ENV=development
```
> After that look like and run flask

```bash
export FLASK_APP=server.py
export FLASK_ENV=development
flask run

```
